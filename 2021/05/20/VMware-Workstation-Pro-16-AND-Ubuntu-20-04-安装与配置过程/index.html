VMware Workstation Pro 16 AND Ubuntu 20.04 安装与配置过程
发表于 2021-05-20| 更新于 2021-05-23|软件安装与配置
阅读量:

钉钉！断了许久未更是吧，并非我偷懒（借口），而是最近开始准备考研了，所以就没时间专业一些技术方面的东西。谁成料到最近走背运…整个电脑的机械硬盘全崩掉。我恨啊！！不幸中的万幸是C盘尚存，许多作业都得以幸存，所以同志们，平日一定要做好数据备份和保存啊。换了新硬盘后就是面对一堆软件的重装和配置…学计算机的真经不起这折腾，前前后后算是搞了一礼拜才把电脑恢复到原来九成的样子，期间也记录了一些软件安装配置的过程，所以首先要写的就是虚拟机的安装配置。

1、下载安装包

首先到官网下载VMware Workstation的安装包及Ubuntu20.04的镜像到自己的电脑上

VMware Workstation下载地址：https://www.vmware.com/cn/products/workstation-pro/workstation-pro-evaluation.html

Ubuntu下载地址：https://ubuntu.com/#download

5、全屏及双向复制的实现

很多人在安装虚拟机后没法实现全屏及双向复制的功能，网上有很多种解决方法，但有的不是过于复杂就是无效，这里提供一条非常简洁的方法，只需要一条命令即可搞定

sudo apt-get install -y open-vm*

新建终端输入该命令后，等待安装完成就会看到系统自动跳着分辨率大小（若没有重启即可）
