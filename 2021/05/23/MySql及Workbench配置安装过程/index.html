MySql及Workbench配置安装过程
发表于 2021-05-23| 更新于 2022-04-03|软件安装与配置
阅读量:

这次我们要安装和配置的MySql，话不多说，上正文

1、下载安装包

第一步我们依旧是上官网Download安装包：https://dev.mysql.com/downloads/mysql/

记住是下载第一个噢

然后我们将安装包解压丢进一个纯英文目录下，比如我将其解压在D:\MySql中

2、配置MySql

然后我们用管理员打开命令行，进入到刚才的目录中，再进入到你解压出来的文件夹下

然后再进入bin文件夹中

安装mysql服务：mysqld --install

出现Service successfully installed.即为成功，然后我们初始化mysql：mysqld --initialize --console，注意记住它所提供的密码！！

然后启动MySql服务：net start mysql

登入：mysql -u root -p，输入刚才系统提供密码，出现下图就成功啦！

当然初始密码太难记，我们可以修改一下密码：alter user 'root'@'localhost' identified by '你的新密码'

修改完记得再登录验证一下

然后我们要设置一下系统变量及环境变量（我的电脑->属性->高级系统设置->环境变量），在系统变量下新建一个变量，变量值你安装包解压出文件夹的位置，然后添加Path变量

3、安装MySql Workbench

安装MySql Workench挺简单的，修改完位置直接无脑next就行

安装完成后重启一下，双击下图红框位置，输入密码，进入界面就可以进行数据库增删改查的可视化操作啦
