DOM的定义

HTML DOM 是 HTML 的标准对象模型和编程接口。它定义了：

  • 作为对象的 HTML 元素
  • 所有 HTML 元素的属性
  • 访问所有 HTML 元素的方法
  • 所有 HTML 元素的事件

通过DOM，我们可以对HTML中的任何属性进行修改

获取元素/结点

我们有两种方法来获取HTML中的标签，一种是通过元素

  • document.getElementById()
  • document.getElementsByClassName()

第二种是通过结点

  • document.querySelector()
  • document.querySelectorAll()
