HTML元素分类

HTML的标签可以分为以下几种元素属性：

  • 块元素：可以设置宽度和高度,独立成行 如：h1-6、p、div、ul、li
  • 行内元素：不可以设置宽度和高度,不独立成行 如：a、span
  • 行内块元素：可以设置宽度和高度,不独立成行 如：img, input, button

当然这些属性并不是固定的，可以通过设置CSS的display属性来实现互相转化

  • display:block 转换为块元素
  • display:inline 转换为行内元素
  • display:inline-block 转换为行内块元素
  • display:none 隐藏元素
