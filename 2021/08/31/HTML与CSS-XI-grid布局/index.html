Grid布局的基本概念

Grid布局将网页划分成一个个网格，可以任意组合不同的网格，做出各种各样的布局，因此我们也称它为网格布局

与Flex布局相同的是，Grid布局同样具有容器和项目两个概念

将元素设置为 display:grid;元素会变为一个flex容器，容器内部的元素我们称为grid项目。

Grid布局下通过网格线划分，n行有n + 1根水平网格线，m列有m + 1根垂直网格线

容器属性

  • justify- Items属性：指定容器内所有项目在单元格的水平对齐方式
  • align-items属性：指定容器内所有项目在单元格的垂直对齐方式
  • justify- content属性：指定所有在单元格在容器的水平对齐方式
  • align- content属性：指定所有在单元格在容器的垂直对齐方式
  • grid-auto-columns属性：设置溢出行的尺寸
  • grid-auto-rows属性：设置溢出列的尺寸

项目属性

  • grid- column- start属性：设置单元格列开始线
  • grid- column-end属性：设置单元格列结束线
  • grid- column: grid- column- start / grid- column-end
  • grid-row- start属性：设置单元格行开始线
  • grid-row-end属性：设置单元格行结束线
  • grid-row: grid-row- start / grid-row-end
  • justify-self属性：指定某个项目在单元格的水平对齐方式
