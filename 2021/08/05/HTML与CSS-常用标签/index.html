VsCode中常用快捷键

  • ! + Tab键：快速生成HTML5头部标签
  • Ctrl键 + /：快速生成注释
  • Shiftl键 + Altl键 + F：对齐代码
  • Alt+Shift+下键：快速复制上一行的代码
