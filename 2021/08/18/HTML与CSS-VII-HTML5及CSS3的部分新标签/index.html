HTML与CSS VII:语义化标签、媒体元素、圆角、阴影与形变
发表于 2021-08-18| 更新于 2021-08-18|HTML与CSS
阅读量:

本篇博客我们讲述HTML5中常用的部分新标签及CSS3中一些常用的新属性

语义化标签

语义化标签本质上还是个div，只不过通过语义化法方式能让代码更加规范，也能让阅读代码的人更加一目了然

  • header：网页头部
  • nav：导航栏
  • aside：侧标栏
  • article：显示文章
  • section：布局的一部分
  • footer：网页页脚

媒体元素

  • audio：音频
  • video：视频

对于媒体元素我们有以下三个常用的控制属性：controls-显示控制面板、autoplay-视频自动播放、muted-静音

圆角边框与阴影

  • border-radius：圆角属性

若后面有四个参数，分别是左上、右上、右下、左下(顺时针)

若后面有两个参数，第一个值表示左上和右下，第二个值表示右上和左下

若后面仅有一个参数，则表示四个角

  • box-shadow：阴影

后接参数分别表示：x轴偏移量、y轴偏移量、模糊半径、阴影颜色(不设置颜色为黑色)
