发表于 2021-08-17| 更新于 2021-08-17|HTML与CSS
阅读量:

本篇博客讲述的是CSS中绝对定位，相对定位和固定定位的内容

CSS定位

css可以给元素设置三种不同的定位效果，分别是

  • 绝对定位: absolute
  • 相对定位: relative
  • 固定定位:fixed

设置定位后，我们可以给元素添加坐标属性(非定位元素不起作用)：top, left, right, bottom, z-index
