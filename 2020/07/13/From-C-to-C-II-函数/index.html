    函数模板

    C++中存在，存在泛型编程的概念，即不考虑具体数据类型的编程方式，函数模板是一种特殊的函数，可以使用不同的类型进行调用，对于功能相同的函数，不需要重复编写代码，并且函数模板与普通函数看起来很类似，区别就是类型可以被参数化。

    Eg：

    include

    using namespace std;
    template

    T add(T x, T y) {
    return x + y;
    }

    int main(){
    cout << add(5,3) << endl;
    cout << add(5.1,3.2) << endl; //也可不加类型让系统判断，但需要同类参数
    system(“pause”);
    }
