From C to C++ I:输入和输出
发表于 2020-07-12| 更新于 2020-07-13|C++
阅读量:

暑假打算学一学虚幻引擎的使用，虚幻引擎所支持的代码是C++，而学校只开了C的课程，就得自学一下C++方面的相关知识

好在C++就是C的进阶版，很多地方和C并无二异，而自己又学过JAVA，所以理解C++的面向对象程序设计应该也不会有什么大碍

但是一些零碎的知识点不常用的话肯定会忘，所以打算写这“From C to C++”一系列博客来帮助我记忆，本篇记录的就是C和C++中输入与输出的不同之处

C++采用输入输出流进行数据的输入和输出

首先需要声明相关语句

C++语法中标准输入语句：cin>>变量1>>变量2>>…>>变量n;

一条输入语句可输入多个变量，输入时用空格或Tab键隔开，以回车结束

Eg：输入x和y

C++语法中标准的输出语句：cout <<表达式1<<表达式2<<…<<表达式n;

一条输出语句可输出多个表达式结果，输出结果间没有间隔，常以”endl”作为换行

Eg：输入一个数，并将其输出
